About Us

Frequently Asked Questions

Will I be able to update the website myself?

Yes! We primarily use WordPress which allow you to easily update your website’s content without needing any coding knowledge, including changing pictures, adding blog posts, and updating content. If you have any questions feel free to reach out to us.

I want _ feature in my site, can you do that?

Probably. There are so many things that we can easily add to your website. If we do not currently know how to do what you want, we are always willing to learn and try new things.

How much does a website cost?

This is a common question with no one-size-fits-all answer. Web design costs can vary depending on the complexity of the website, the features you need, and if you would like us to upkeep the website for you.
